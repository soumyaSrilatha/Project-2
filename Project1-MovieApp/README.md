# Movie Review App

## Tech Stack
- Frontend: React.js
- Backend: Node.js + Express
- Database: MongoDB

## Features
- User authentication
- CRUD for movie reviews
- Star ratings and comments

## Folder Structure
- frontend/  → React app
- backend/   → Node.js API
- api-docs/  → API documentation and screenshots

## How to Run

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
cd frontend
npm install
npm start
```
